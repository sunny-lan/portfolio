Leilo - A simple IoT platform

Built with: PHP/MySQL, ReactJS, C++ and ESP8266
Leilo is a simple, open sourced, platform that lets users to connect sensors to the internet. Leilo is composed of 3 components:
  • Sensor & sensor software - The sensor hardware is built around the ESP8266, an Wifi-enabled chip. The software regularly reads and uploads sensor values to the API. Sensors are interchangeable as the sensor board has a simple sensor interface.
  • API/backend - The backend is written in PHP and stores sensor values in a MySQL database. The API uses a simple JSON format to transfer data. A NodeJS WebSocket overhaul of the backend is being currently being written.
  • Webapp user interface - Made with ReactJS, the frontend receives live updates from the server, and displays data on a dashboard. The dashboard is easily customizable, allowing users to drag and drop customized widgets. In addition, the complete configuration of the dashboard remains synchronized across browsers.
