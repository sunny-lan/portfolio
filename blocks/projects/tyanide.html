Tyanide - TeachAssist client

Built with: Java 8, JavaFX, JSoup
Tyanide is a desktop application that allows students to access their marks without the fuss of opening a browser and logging into TeachAssist.
  • Notifications - Tyanide detects when a teacher adds, updates, or removes a mark, subsequently notifying you of that change. Tyanide can be set to run in the background, while still displaying notifications.
  • Session saving - Tyanide can save marks and TeachAssist credentials locally, avoiding the fuss of retyping a password every time, and also allowing you to view marks offline.
  • Mark simulation - Tyanide can simulate what average you would get if you got a certain mark on an assignment. Simply double-click the mark you want to change, and type in the hypothetical mark - Tyanide will update your average accordingly (this obviously doesn't change your real mark).
